__toString()

__toString() : string

Cette méthode permet de faire "echo $article".

Ainsi, pour "afficher" $article, PHP affichera en réalité le retour de cette méthode.
Ici, le nom, donc "echo $article" est équivalent à "echo $article->getName()"
