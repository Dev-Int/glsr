Cette méthode permet de faire "echo $supplier".

__toString() : string

Ainsi, pour "afficher" $supplier, PHP affichera en réalité le retour de cette méthode.
Ici, le nom, donc "echo $supplier" est équivalent à "echo $supplier->getName()"
